# NaKamaDemo_Unity

这是一个基于 Unity 的 Nakama 游戏服务器集成演示项目。本项目展示了如何在 Unity 中使用 Nakama 进行后端服务的连接、数据存储与交互。

## 📚 项目文档

项目中包含了一些关于 Nakama 架构设计、生产环境部署以及安全相关的文档，存放在 `docs/` 目录下。您可以点击以下链接查看详细内容：

- 🏗️ **架构与设计**
  - [DDD 弱网环境下的存储设计](docs/ddd-weak-network-storage.md)

- 🔒 **数据与安全**
  - [Nakama 存档数据安全指南](docs/nakama-save-data-security.md)

- 🚀 **部署与运维**
  - [Nakama 生产环境最佳实践](docs/nakama-production-best-practices.md) （[PDF 版本](docs/nakama-production-best-practices.pdf)）
  - [Nakama Ubuntu Docker 部署最佳实践](docs/nakama-ubuntu-docker-best-practices.md)

## 🎮 快速开始

1. 使用 Unity Hub 打开本项目。
2. 确保您已经在本地或远程部署了 Nakama 服务器。
3. 在项目中配置您的 Nakama 服务器地址及相关凭证。
4. 运行 `Assets/Scenes/SampleScene.unity` 或其他 Demo 场景进行测试。
